# HolaTky
A simple hello world gem that generates UUIDs using SecureRandom.
This gem was created as part of a personal learning project. There is no plan for active maintenance.

## Usage

```ruby
HolaTky.generate
=> "f779fd5c-8e1c-4a32-bb1d-407b79588b59"
```
